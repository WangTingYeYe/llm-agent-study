# llm-agent-study

大模型应用开发学习项目

## 项目简介

这是一个专注于大模型应用开发的学习项目，包含多个AI应用实例和最佳实践。

## 项目列表

### 🌍 [旅行规划助手](./旅行规划助手/)

基于 Agno 和 Gradio 构建的AI旅行规划助手，使用 DeepSeek 模型和百度搜索API，能够自动生成详细的旅行计划。

**技术栈:**
- Agno (多智能体框架)
- DeepSeek (大语言模型)
- Gradio (Web界面)
- BaiduSearch (搜索工具)

**功能特点:**
- 🤖 智能规划生成
- 🔍 实时信息搜索
- 📋 完整行程安排
- 🎨 友好用户界面

## 技术栈

- **AI框架**: Agno
- **大模型**: DeepSeek
- **界面框架**: Gradio
- **开发语言**: Python 3.10+
